# Brain‑24 Documentation  
System Architecture • Cognitive Model • Engineering Guides

This directory contains all documentation for the **Brain‑24 architecture**, including subsystem posters, cognitive models, memory systems, skills, consolidation, and the control plane.  
It is the central reference for understanding, extending, and maintaining the Brain‑24 system.

# 2. What’s Inside `brain-24/`

The Brain‑24 documentation is organized into the following major areas:

### 🧠 Cortex (C1–C5)
Immediate cognition, meta‑cognition, self‑directed cognition, tool‑augmented cognition, reflective cognition.

### 🧩 Memory Organ
Episodic, semantic, procedural memory, and cross‑memory interactions.

### 🛠 Skills Organ
Skill storage, execution, evaluation, and confidence updating.

### 🔄 Consolidation Engine
Experience → knowledge → skills transformation.

### 🧭 Control Plane
Router, Orchestrator, and the unified control plane.

### 🔗 Cross‑Organ Learning
The full adaptive learning loop across C2, Skills, Memory, Consolidation, and C5.
